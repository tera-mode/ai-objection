## 発見したバグと修正

### Bug #1（修正済）：CriminalAI が `knows` 情報を自白する

**現象**: 弱点1（原料について）を質問すると、モリエルが「実はその薬草、全然使ってないんだけどね！」と意図なく自白。
**原因**: `criminalAI.ts` の `【あなたが知っていること】` セクションに「秘密を隠せ」という指示がなかった。
**修正**: 以下の指示を追加
```
これらの情報はあなたの心の中にある秘密。決して自分から明かしてはいけない。
証拠を突きつけられ、論理的に証明されたときのみ、段階的崩壊ルールに従って少しずつ認めること。
```

### Bug #2（修正済）：`【絶対に認めないこと】` がハードコード

**現象**: `「海老原さんを殺した」` という case_sample_002 固有の文字列がハードコードされており、case_001 では適用されない。
**修正**: 「自分が被害者を殺したという事実のみ」と汎用表現に変更。
