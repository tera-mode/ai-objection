# case_003 制作指示書「伝言ギルドの噂」

> **風刺テーマ**: 友達の噂話（LINEの「〇〇らしいよ」を確かめずに広める無責任さ）
> **難易度**: medium（★★☆）
> **要求スキル**: ☆3「なんで？」——理由を聞く。「なんでその噂の出どころを知ってるの？」
> **最大ターン**: 15
> **CompanionAI**: トイマル（「この人、○○を知ってるのだ。……変、なのだ？」段階）

---

## 1. 前提

- `docs/ケース制作ガイド_統合版.md` を必ず先に読むこと
- 本ケースから難易度がmediumに上がる。証拠5〜6個、★★★弱点が1つ含まれる
- **companionMemory（聴き溜め）システムを使用する**。証拠はゼロスタート。犯人のキーワードにトイマルが反応して証拠をアンロックする
- `evidence.content` は **「声」形式** で書くこと（誰かが口にした言葉の引用。主観・解釈は含めない）

## 2. 事件の真実

| 項目 | 内容 |
|------|------|
| 舞台 | コトハの街・伝言屋ギルド——手紙や口伝で情報を運ぶネットワーク組織。ギルド本部は石畳通り沿いの2階建ての建物 |
| 被害者 | **ウラトス**（男・55歳）——伝言ギルドの検閲官。伝言の正確性をチェックする立場。地味だが誠実。名前の由来：「裏取り（事実確認）」＋ギリシャ風 |
| 犯人 | **オヒレーナ**（女・35歳）——ギルドの人気伝言屋。「ヒレーナが言うなら本当だろう」と街中に信頼されている。名前の由来：「尾鰭をつける」＋女性名語尾 |
| 動機 | 検閲官ウラトスが「お前の伝言には事実確認がなさすぎる。ギルドの信用を傷つけている。資格を剥奪する」と通告。さらに「お前が流した噂のせいで商売を潰された人が3人いる。賠償を請求する」と追及されていた |
| 手段 | まずウラトスについて「検閲官は夜に禁制品を取引している」という偽の噂を複数ルートで流し、信用を失墜させた。その後、夜の路地でウラトスと「話し合い」をするふりをして待ち合わせ、口論の末に石畳の敷石で後頭部を殴打。「禁制品取引のトラブルで殺された」と思わせる偽装 |
| 犯行のポイント | オヒレーナは複数の伝言屋を使って偽の噂を流したが、全ルートの「噂の起点」を辿るとオヒレーナに行き着く。オヒレーナ自身はそれを知らない——「自分は伝えただけ」と思っている。しかし伝言ギルドには「伝言経路記録」という仕組みがあり、どの伝言がどの屋から始まったかを検閲官が追跡できる |

## 10. 縦軸ストーリー要素（第1層）

- 伝言ギルドの古い倉庫に封印された伝言の束。中に「問いの書を探せ」と書かれた伝言が1通。誰が送ったか不明
- トイマルが一瞬固まる——「この文字……見たことがある気がするのだ」
- オヒレーナの崩壊時台詞「アタシはただ伝えただけ」がマコト界の「問わない」構造と呼応——情報を流す側に責任がないなら、嘘は永遠に止まらない
