# case_002 制作指示書「闘技場の不正」

> **風刺テーマ**: 強い奴＝正しい（声がデカい・力が強い＝正しいと思い込む。スクールカースト）
> **難易度**: easy（★☆☆）
> **要求スキル**: ☆1「ん？」——なんか変だな、と気づく
> **最大ターン**: 15
> **CompanionAI**: トイマル（イバルドにビビるが、なのの横に立ち続ける）

---

## 1. 前提

- `docs/case_dev_guide.md` を必ず先に読むこと
- `data/cases/case_sample_002.json`（水族館ケース・hard）を参照実装として参考にすること
- case_002 は **2番目のケース**。case_001で「饒舌な犯人の情報洪水」を体験した後、今回は「威圧で黙らせようとする犯人」。対照的なタイプ
- 世界観は **マコト界（中世ヨーロッパ風ファンタジー）**。時刻は「鐘の数」で表現する
- ☆1「ん？」がテーマ。カバーストーリーの中に**単純だが見逃しやすい矛盾**を仕込む

## 2. 事件の真実

| 項目 | 内容 |
|------|------|
| 舞台 | コトハの街の闘技場——ボードゲームと体術の複合競技が行われる大型施設。控え室・練習場・観客席・審判室がある |
| 被害者 | **ハカール**（女・65歳）——闘技場の女性審判。小柄だが毅然とした態度で公正なジャッジをする。25年間無欠勤 |
| 犯人 | **イバルド**（男・30歳）——闘技場の無敗チャンピオン。街のヒーロー。子供たちの憧れ |
| 動機 | ハカールが「イバルドは禁止技（関節の逆固め）を常習的に使っている」という証拠を集めていた。無敗記録は不正の上に成り立っている。次の大会で告発する予定だった |
| 手段 | 「練習の相手をしてほしい」と練習場にハカールを誘い出し、「組み手の練習中に手加減したつもりが当たりどころが悪かった」と事故に偽装。実際は背後から頭部を打撲させた |
| 犯行のポイント | 練習場は通常、鐘8つ（夜）に施錠される。イバルドはチャンピオン特権で鍵を持っており、施錠後に入れる。しかし「鐘7つに帰宅した」と主張。練習場の施錠記録とイバルドの鍵の使用痕跡が矛盾する |

## 9. 完成前チェックリスト（ケース制作ガイド準拠）

```
[ ] timeline を読んで、イバルドの行動に矛盾がないか
[ ] knows に「timeline でイバルドがやったこと」が全部含まれているか
[ ] doesnt_know の各項目が timeline から証拠として観測可能か
[ ] knows と doesnt_know に同じキーワードが出ていないか
[ ] structural_weaknesses の各弱点が evidence の証拠と対応しているか
[ ] evidence の各 id が structural_weaknesses から正しく参照されているか
[ ] claim に「後で崩れる証拠」それぞれへの嘘の準備が含まれているか

[ ] 各証拠の content に「AIが逃げ道を作れない補足情報」が含まれているか
[ ] surface に具体的な口癖（フレーズ）が含まれているか
```

## 10. 縦軸ストーリー要素（第0層）

イベントやstoryTextの中に以下の手がかりを自然に埋め込むこと：

- 闘技場の壁に古い碑文「力なき言葉にも——」（途中で削られている）
- イバルドが「あんなもん昔の落書きだ」と一蹴するが、トイマルが碑文をじっと見つめる
- トイマルがイバルドの威圧に怯えながらも「な、なの……怖くないのだ？」と聞く。なの「怖いよ。でも、怖いからって黙ったら——ハカール先生と同じことが繰り返される」
- イバルドの崩壊時に「強い奴が正しい」の論理が崩れる＝マコト界の「問わない＝安全」と根は同じであることが暗示される
