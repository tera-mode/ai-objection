# 第1章　ゲームの核心

## 1-1. タイトル

> **問わない異世界の探偵少女**

## 1-2. ワンライナー

> 「問わない異世界」に転生した普通の中学生が、「なんで？」と聞くだけで英雄になる推理ゲーム。

## 1-3. テーマ

**「問う力」—— 受け取った情報に対して、自分の頭で疑問を持ち、考え、確かめる力。**

このゲームをクリアした子供が、ショート動画を見た時に「ちょっと待って。なんで？ 本当に？」と3秒だけ立ち止まれるようになったら、このゲームは成功。

## 1-4. ゲームシステム概要

AIが演じる犯人に対して、プレイヤーが自由にテキストで質問する尋問ゲーム。3つのAIが連携して動く。

| AI | 役割 | 技術 |
|---|---|---|
| **CriminalAI** | 犯人を演じる。嘘をつき、追い詰められると段階的に崩壊する | Gemini API |
| **JudgeAI** | プレイヤーの質問が犯人の矛盾を突いているか判定する | Gemini API |
| **CompanionAI（トイマル）** | 相棒。証言の記録、感情的リアクション、ヒントの示唆 | Gemini API |

## 1-5. コアループ

```
事件概要の提示（CrimeSceneフェーズ）
  ↓
尋問開始（Interrogationフェーズ・最大15ターン）
  ↓
  プレイヤーが自由に質問を入力
    → CompanionAI（トイマル）がリアクション
    → CriminalAI が返答
    → JudgeAI が矛盾判定
    → コヒーレンス（犯人の動揺度）に反映
    → CompanionAI がフォローコメント
  ↓
逮捕成功 or タイムアップ（Resultフェーズ）
```

# 第2章　世界観：マコト界

## 2-1. 基本設定

マコト界は**中世ヨーロッパ風のファンタジー世界**。木骨造りの建物が並ぶ石畳の街、馬車が行き交う広場、ギルドや市場、教会のような「マコト堂」。魔法は存在しないが、「コトバ」に特別な力があるとされる文化を持つ。

住人は人間とトイマルのような亜人種が共存する多種族社会。技術レベルは中世〜近世で、紙と羽ペン、蝋燭と石畳が日常。ただし「コトバの記録装置」（トイマル族の聴き役が担う）など、独自の文明要素がある。

唯一にして決定的な特徴は、住人が情報を受け取った時に**「なんで？」「本当に？」と問う感覚を持っていない**こと。

### 世界の見え方

| 領域 | マコト界の実態 | 現実世界との対応 |
|---|---|---|
| 教育 | マコト堂の教師の言葉は絶対。質問は「不敬」 | 「空気を読め」の同調圧力 |
| 商売 | 「万病に効く薬草です」と言えば売れる。効かなくても自分を疑う | 誇大広告を鵜呑みにするSNS消費 |
| 裁判 | 証言がそのまま真実として採用される。物証より言葉が重い | 権威への盲信 |
| 人間関係 | 噂が検証なしに街中に広まる。口伝が事実になる | SNSでの裏どりなき拡散 |
| 犯罪 | 犯人の嘘が通用するのは巧みだからではなく、誰も問わないから | AIの出力をそのまま会議に出す |

### 世界の風景（キービジュアル準拠）

- **街並み：** 木骨造り（ハーフティンバー）の2〜3階建ての建物。赤茶色の屋根瓦。花飾りの窓。石畳の路地
- **市場広場：** 木樽、木箱、布屋根の露店。果物や香辛料、怪しい薬草
- **マコト堂：** 街の中心にある教会風の建物。「マコトの誓い」の石碑がある
- **衛兵詰所：** 尋問が行われる場所。石造りの堅牢な建物。蝋燭の灯り

### 世界のルール（ゲームメカニクスとの接続）

| ルール | ゲーム上の機能 |
|---|---|
| 言葉を額面通りに受け取る | 犯人のカバーストーリーが「普通に通用している」状態からスタート |
| 「問われる」経験がない | コヒーレンスが下がりやすい→テンポよく崩壊 |
| 矛盾を突かれると周囲も気づき始める | JudgeAIの矛盾検出＝世界の信頼構造が揺らぐ |
| 嘘が完全に暴かれると社会的に致命的 | 崩壊＝この世界における最大の禁忌の露呈 |

## 2-2. マコトの誓い

> **「問うな。信じよ。さすれば心安らかなり。」**

マコト堂の壁、街の広場の石碑、学校の黒板——あちこちに刻まれた標語。子供の頃から繰り返し唱えさせられる。マコト堂では「マコト学」の授業があり、「人の言葉を信じる練習」を行う。

### 公式の歴史（表向き）
かつてマコト界は「疑いの時代」と呼ばれる戦乱期にあった。誰もが誰かを疑い、王国は崩壊しかけた。初代大裁定官が「マコトの誓い」を定め、以来平和になった。

### 真実（ストーリー後半で開示）
「疑いの時代」に壊されたのは王国ではなく、**権力者の嘘**だった。問う者たちが不正を暴いたことを恐れた権力者が「問うこと＝悪」を法にした。

## 2-3. コヒーレンスメーターの世界観的意味

**「コトバの鎧」——問われなかった安心が崩壊する。**

マコト界では、言葉に力がある。犯人が語る嘘は視覚的に「鎧」のように本人を包んでいる。なのが矛盾を突くたびに鎧にヒビが入り、最後は砕け散る。

# 第3章　主人公：南出 なの（みなみで なの）

## 3-1. 基本プロフィール

| 項目 | 内容 |
|---|---|
| 名前 | **南出 なの（みなみで なの）** |
| 年齢 | 14歳（中学2年） |
| 元の世界 | 現代日本の普通の中学生 |
| 転生の経緯 | 図書室で推理小説を読んでいたら異世界に飛ばされた（最終ケースで回収される伏線） |
| 特殊能力 | なし。現代日本で育った「普通の感覚」がこの世界では異能に等しい |

## 3-2. 名前の由来（ダジャレ命名）

フルネームを声に出すと**「なんで なの？」**。

- **「南出（みなみで）」→「なんで」**：ゲームのテーマそのもの
- **「なの」**：「本当**なの**？」「嘘**なの**？」——疑問の語尾が名前
- マコト界の住人は「なんで」という問いに馴染みがないため、名前の真の意味に誰も気づかない

## 3-3. 外見（キービジュアル準拠）

- **髪：** ダークブラウンのボブカット。肩にかかるくらいの長さ。毛先が軽く外ハネ
- **目：** 大きな琥珀色の目。好奇心と少しの不安が同居する表情
- **服装：** 現代日本のセーラー服（紺色ブレザー、赤いリボン、ベージュのプリーツスカート、紺のハイソックス）をそのまま異世界に持ち込んでいる。マコト界の住人から見ると「変わった異国の衣装」
- **体格：** 中学生らしい華奢な体格。犯人と向き合う時に体の小ささが際立つ
- **特徴的ポーズ：** 核心を突く質問の前に指を突きつける（キービジュアルのポーズ）

### セーラー服の設定上の意味

なのが転生時に着ていた制服をそのまま着続けている。マコト界には存在しないデザインなので、街を歩くと目立つ。これが「この世界の人間ではない」ことの視覚的マーカーになる。着替えを勧められても「あたしの世界の服だから」と着続ける＝元の世界への未練と、自分のアイデンティティへのこだわり。

## 3-4. 性格

- 推理漫画が好きで探偵に憧れているが成績は中の下
- 正義感は強いがビビり。でも不正を見過ごせず、結局突っ込んでしまう
- 時々カッコつけた探偵風の台詞を言ってトイマルに「似合わないのだ」と言われる
- 異世界の常識がわからず恥をかくことも多い（食べ方、挨拶、通貨など）

## 3-5. 口調

- 一人称：**「あたし」**
- ツッコミ担当。「ちょっと待って、それおかしくない？」「いやいやいや、嘘でしょ！？」
- 緊張すると早口。重要な指摘の前に「……あのさ」と一拍置く
- 決意の瞬間：指を突きつけて「——聞かせてもらっていい？」

## 3-6. なぜ尋問するのか

マコト界で殺人が起きた時、衛兵も裁定官も犯人の言葉を疑えない。唯一「なんで？」と問えるなのだけが事件を解決できる。最初の事件は巻き込まれ型。以降は「問える者」として衛兵団から非公式に協力を求められる。

## 3-7. 表情バリエーション

| 状態 | 表情 | 使用場面 |
|---|---|---|
| 通常 | 明るい笑顔。好奇心旺盛な目 | 日常パート |
| 気づき | 目が見開く。髪がぴょんと跳ねる | 矛盾を発見した瞬間 |
| ツッコミ | ジト目。半目 | トイマルや異世界人のボケに |
| 緊張 | 汗。唇を噛む。でも目は逃げない | 犯人と対峙する場面 |
| 決意 | 指を突きつける。目がキリッと変わる | 核心を突く質問の直前 |

# 第4章　相棒：トイマル

## 4-1. 基本プロフィール

| 項目 | 内容 |
|---|---|
| 種族 | マコト界の「聴き役」。人の話を聴いて記録する役割を持つ亜人種 |
| 大きさ | なのの腰あたりまでの身長。小さな子供くらい |
| 能力 | 人の言葉を一字一句記憶する。ただし「問う」回路がない |

## 4-2. 名前の由来（ダジャレ命名）

- **「問い」＋「丸呑み」**＝問いを丸ごと持っていない存在
- 最終ケースで初めて「なんで？」と問う→「問いを丸ごと持つ者」に反転

## 4-3. 外見（キービジュアル準拠）

- **体型：** 小柄でずんぐりした二頭身寄りの体型。愛らしいマスコット体型
- **耳：** 大きな狐のような耳。ミントグリーン色。内側はやや薄い色。「聴く」がテーマなので耳が最大の特徴
- **毛色・髪：** ミントグリーン（薄い青緑）の髪と毛並み。ふわふわした質感
- **目：** 大きな緑色の瞳。丸くて表情豊か
- **服装：** ミントグリーンのオーバーサイズのコート（ダッフルコート風）。コートの前ボタンにドクロ模様のバッジ。黒いインナー、黒いブーツ
- **尻尾：** ふわっとした狐のような尻尾（コートの裾から覗く）
- **手足：** 短くて小さい。物を掴めるが不器用

### デザインの設定上の意味

トイマルの「聴き役」族は、マコト界の中でも独特の存在。大きな耳は「全てを聴く」種族としての進化の結果。ミントグリーンの体色は街の住人と明確に異なり、「人間ではない存在」であることが一目でわかる。オーバーサイズのコートは「体より大きな役割を背負っている」暗示。

## 4-4. 口調

- 一人称：**「ボク」**（性別不明）
- 語尾：**「〜なのだ」**

| 場面 | 台詞例 |
|---|---|
| 通常 | 「ボクの耳は全部聞いてるのだ！」 |
| 証言引用 | 「あの人はさっき『○○』と言ったのだ」 |
| 混乱 | 「えっ？ ○○と△△、両方本当じゃダメなのだ？」 |
| 感動 | 「なの、すごいのだ……！」 |
| 怖い | 「な、なの……この人、怖いのだ……」 |

## 4-5. ギャップ設計（3層）

1. **偉そう ↔ 無知：** 「ボクが案内してやるのだ！」→ 実は何もわかっていない
2. **記録力は完璧 ↔ 理解力はゼロ：** 矛盾した情報を「どっちも本当なのだ」と並列保存
3. **先輩のつもり ↔ 守られている：** 犯人が怒鳴るとなのの後ろに隠れる

## 4-6. 真実（最終ケースで開示）

「聴き役」族はかつて**「問いの聴き手」**だった。問う者の傍で言葉を記録し矛盾を照合する役割。「マコトの誓い」以降、「聴くだけで問わない」種族に変えられた。

## 4-7. トイマルの「正直な嘘」

| 台詞 | 初見 | 真実を知った後 |
|---|---|---|
| 「ボクは嘘がわからないのだ」 | そういう種族 | 「わかっちゃいけない」と教えられた |
| 「この世界では昔からそうなのだ」 | 元々こう | もっと昔は違った |
| 「疑うって……怖い言葉なのだ」 | 文化の違い | 「怖い」と教育された結果 |
| 「なのは特別なのだ」 | 異世界人だから | 洗脳されていないだけ |

# 第5章　「問う力」スキルフレームワーク

## 5-1. 5段階のスキル階段

| レベル | 問い | ゲームでの発動 | 現実で身につく力 |
|---|---|---|---|
| ☆1 | **「ん？」** なんか変だな | 証言の単純な矛盾を見つける | ショート動画で引っかかる |
| ☆2 | **「ちょっと待って」** すぐ信じない | 饒舌な犯人に流されない | RTする前に考える |
| ☆3 | **「なんで？」** 理由を聞く | 「なぜそれを知っているの？」と聞く | AIの回答を鵜呑みにしない |
| ☆4 | **「本当に？」** 確かめる | 複数の証拠を突き合わせる | 別のソースで裏を取る |
| ☆5 | **「つまり？」** 自分で考える | 推論の鎖を自力で完成させる | 自分の意見を自分の言葉で持つ |

## 5-2. 学術的根拠

| スキル | 根拠 |
|---|---|
| ☆1 | ソクラテス「明確化の問い」/ SIFTメソッド「Stop」 |
| ☆2 | SIFT「Stop」/ 下村健一「まだ分からないよね」（小学校教科書掲載） |
| ☆3 | ソクラテス「前提の問い」/ グロービスCT「Why So?」 |
| ☆4 | SIFT「Investigate / Find」/ グロービスCT「True?」 |
| ☆5 | ソクラテス「結果の問い」/ グロービスCT「So What?」 |

# 第6章　ケース構成と難易度曲線

## 6-1. 全体構成

既存の現代日本ケース（動物病院・忍者テーマパーク・水族館）は全て没。マコト界のファンタジー世界に合わせた新規ケースを設計する。

| 2 | 宿屋の密室殺人 | 石畳通りの宿屋 | ★☆☆ | ☆2「ちょっと待って」 | 壁の消された碑文 |
| 3 | 鍛冶師の弟子殺し | 職人街の鍛冶工房 | ★★☆ | ☆3「なんで？」 | 古い書庫の禁書 |
| 4 | 薬草園の園長殺害 | 街外れの薬草園 | ★★☆ | ☆4「本当に？」 | 「マコトの誓い」開示 |
| 5 | 劇団座長の転落死 | 街の劇場 | ★★★ | ☆5「つまり？」 | 長老の公式歴史 |
| 6 | 商会長の毒殺 | 大商会の屋敷 | ★★★ | ☆4+☆5 複合 | 公式歴史への疑問 |
| 7 | 裁定官補佐の殺害 | マコト堂の内部 | ★★★ | ☆1-5 統合 | 受益者の存在 |
| 8 | 最終ケース | 大裁定官の居城 | ★★★★ | 全＋世界への問い | ルミーナ重体→生存。真実の全貌。トイマル覚醒 |

## 6-2. ケース設計方針

### ファンタジー世界ならではの設計

- **物証が中世的：** 防犯カメラの代わりに「門番の記録簿」「夜警の巡回報告」「聴き役の記憶」
- **科学捜査の代わりに：** 薬師の毒物鑑定、鍛冶師の金属分析、聴き役の証言記録
- **アリバイの構造：** 馬車の移動時間、城門の開閉記録、鐘楼の時報による時刻特定
- **殺害手段：** 毒殺（薬草の知識）、刺殺（鍛冶の武器）、転落（建造物の構造）など中世的手段

### 各ケースに必要な要素（ケース制作ガイド準拠）

```
1. timeline：時系列の全事実
2. criminal.personality：3段階の口調変化（surface / underPressure / collapsed）
3. criminal.knowledge.knows：犯人が知っていること
4. criminal.knowledge.doesnt_know：犯人が知らないこと
5. criminal.coverStory.claim：最初の嘘
6. criminal.coverStory.structural_weaknesses：弱点（証拠不要1つ＋証拠連動複数）
7. evidence：物証リスト（AIが逃げ道を作れない補足情報込み）
8. storyText：intro / criminalIntro / victory / defeat
```

## 6-3. ケース1「市場の毒殺事件」（チュートリアル）概要案

| 項目 | 内容 |
|---|---|
| 舞台 | マコト界の首都「コトハの街」の市場広場 |
| 被害者 | 市場で人気の果物商人。誰からも愛される老人 |
| 犯人 | 隣の露店の香辛料商人。被害者の人気に嫉妬 |
| 手段 | 被害者が毎朝飲む薬草茶に毒草を混入 |
| 矛盾の核心 | 犯人は「朝は自分の店の準備で忙しく、被害者の店には近づいていない」と言うが、「今朝の薬草茶はいい香りだった」と匂いを知っている。被害者の店でしか淹れない茶の香りを知っているのはおかしい |
| 要求スキル | ☆1「ん？」——「近づいていない」のに「香りを知っている」という単純な矛盾 |
| 証拠 | 門番の記録簿（開門時刻と入場者）、薬師の毒物鑑定書、目撃者の証言 |

# 第7章　縦軸ストーリー：段階的開示

## 7-1. 設計原則

「公平な反転の4段階」に基づく。各ケース単体で楽しめること。プレイヤーの10〜15%が最終開示前に気づくバランス。

## 7-2. 開示スケジュール

### 第0層（ケース1-2）：表面の印象

「不思議な異世界だけど面白い」。

**埋め込む手がかり：**
- マコト堂の壁に「今日のマコト：問うな、信じよ」と書かれた掲示板
- 古い石壁に薄れた彫り文字「疑え、さもなくば——」（途中で削られている）
- トイマルが「昔のことは……よくわからないのだ」と口ごもる

### 第1層（ケース3-4）：違和感の蓄積

なのが気づく：「質問するだけで周りが凍りつく。これは普通じゃない」

**発見：** 古い書庫の奥に「問いの書」という封印された本。中身は空白だが、ページの端に消されかけた文字「問え。考えよ。自らの——」

### 第2層（ケース5-6）：「マコトの誓い」の公式説明

マコト堂の長老が歴史を語る。「かつて問い続ける者たちが王国を壊した。だから初代大裁定官が『問うな、信じよ』を定めた」

なのの内心：「でも……問わなかったら、嘘も間違いも止められないじゃん」

### 第3層（ケース7）：「問わない社会」の受益者

マコト堂の内部で事件が起きる。裁定官補佐が殺され、犯人は権力に近い人物。追い詰められた犯人が「お前がしていることは、マコトの誓いに反する犯罪だ」と宣言。

なのが気づく：「この人、嘘がうまいんじゃない。周りが誰も聞かないだけだ」

### 第4層（最終ケース）：真実の全貌

大裁定官の居城で最後の事件。ルミーナが重体から生還し、真実が開示される。トイマル覚醒。

> **トイマル**「なんで……ボクは今まで『なんで？』って言えなかったのだ？」

# 第10章　エンディング

トイマルが初めて「なんで？」と問い、7つ目の問いの星がトイマル自身の星として灯る。

なのが画面に向かって：

> 「マコト界のみんなは、問うことを忘れてた。『信じることが優しさ』って教えられて、誰も『なんで？』って言わなくなった。 ——ねえ、あなたの世界はどう？ 動画で見たこと、そのまま信じてない？ AIが言ったこと、確かめてる？ ……べつに、全部疑えって言ってるんじゃないよ。ただ——**『本当かな？』って、一回だけ立ち止まってみて**。それだけで、世界はちょっと変わるから。」
