# case_004 制作指示書「月夜の唄の歌姫」（完全版）

> **風刺テーマ**: 推しの表裏（アイドル・推しの「表の顔」を信じきること。美しい人は嘘をつかないと思い込む）
> **難易度**: medium（★★☆）
> **要求スキル**: ☆3-4「なんで？」＋「本当に？」——同情に流されず事実を確かめる

## 2. 事件の真実

| 項目 | 内容 |
|------|------|
| 舞台 | コトハの街・酒場「月夜の唄」——音楽酒場。1階が舞台と客席、2階が特別個室（VIPパトロン用）と楽屋。舞台袖から2階への裏階段がある（客席からは見えない） |
| 被害者 | **カネモス**（男・55歳）——歌姫の熱狂的パトロン。裕福な商人。名前の由来：「金を守る」。最初は純粋な支援者だったが次第にストーカー化。「歌を辞めて自分だけのものになれ。さもなくばお前の秘密をバラす」と脅迫していた |
| 犯人 | **カブリーナ**（女・28歳）——街一番の歌姫。名前の由来：「猫を被る」。美しく、街中にファンがいる |
| 動機 | 二重の動機。①カネモスのストーカー行為から逃れたい（同情できる動機）。②カネモスが「カブリーナが祖母から受け継いだ薬草の知識で禁制薬草の密売をしていること」を知っており、「衛兵に通報する」と脅迫していた（同情しにくい動機）。当日、カネモスが「もう待てない。明日、衛兵に全部話す」と最後通牒を突きつけた |
| 手段 | 公演の**間奏パート**（楽士が演奏する約5分間、歌姫は舞台袖に下がる慣習）の間に、舞台袖の裏階段から2階に上がった。特別個室で酒を飲んでいたカネモスの背後からリュートの金属弦で絞殺。遺体を窓際に引きずり、窓を開けて裏路地に落とし「酔って窓から転落した事故」に偽装。窓を閉め、凶器の弦を楽屋のリュートケースに戻し（1本だけ新品に張り替え）、裏階段から舞台袖に戻って後半の歌を何事もなかったように歌い上げた |
| 犯行のポイント | ①リュートの弦は歌姫が常に持ち歩く消耗品で、凶器として不自然ではない。②間奏パートは毎公演の恒例で、歌姫が舞台袖に下がるのを客は全員知っている。「ずっとステージにいた」は嘘だが、「間奏中も含めて公演中だった」は事実なので、アリバイが成立しているように聞こえる。③特別個室の窓は腰の高さで、酔っ払いが自分で身を乗り出すには低すぎない微妙な高さだが、実際には意図的に押し出さないと落ちない構造 |

## 8. 感情的アンカー

カネモスの懐から出てきた自画像。歌姫の歌を聴いて泣いている下手な絵。歪んだ愛だったが、歌に救われた瞬間は本物だった。加害者と被害者の関係が単純ではないことを示す。

introで登場（「お世辞にも上手とは言えない絵」）→ victoryで回収（「歪んだ愛だったけれど、歌に救われたことは本当だった」）。

## 10. トイマルの特別な体験

このケースでトイマルにとって決定的な体験が起こる。

カブリーナの涙に完全に同情する。「かわいそうなのだ……」「この人は悪くないのだ……」と本気で信じる。なのが証拠を突きつけようとするたびに「なの、この人を傷つけちゃダメなのだ……」と止めようとすらする。

しかし弱点6のトドメでカブリーナの仮面が落ちた瞬間、トイマルは衝撃を受ける。

> **トイマル**「えっ……嘘……なのだ？ あの涙は……全部……」
> **なの**「……トイマル。泣いてるから正しいとは限らないんだよ」
> **トイマル**「……ボク、信じちゃったのだ。見た目がきれいで、泣いてて、かわいそうだったから。……でもそれは、理由にならないのだ？」
> **なの**「ならない。事実を見なきゃ」

**これがトイマルの成長の転機。** 「見た目で信じちゃダメ」「涙を流しているから正しいとは限らない」を身をもって体験する。case_005以降、トイマルは感情に流されそうになったとき「待つのだ……前にこんなことがあったのだ」と自分でブレーキをかけ始める。

## 11. 縦軸ストーリー要素（第1層）

- 酒場の地下室（倉庫）に古い楽譜が見つかる。曲名は「問いの歌」。歌詞は消されているが、メロディの譜面だけ残っている
- カブリーナは「知らない曲ね」と言う
- トイマルの耳がピクリと動く——「この旋律……ボク、知ってる気がするのだ。ずっと昔に聴いたような……でも思い出せないのだ」
- 楽譜の端に小さな記号。聴き役族の古い紋章に似ている
- カブリーナの崩壊時台詞「推される側の気持ち、わかる？」がマコト界の「問わない」社会の構造と呼応——問わない社会は、表の顔を維持し続けなければならない人も苦しめている。「仮面を被り続けること」の苦しみは、この世界の構造的な問題でもある
