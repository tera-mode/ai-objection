# case_007「マコト堂の恩師」

> **風刺テーマ**: 先生＝正しい（教科書・校則・大人の「常識」を疑えない）
> **難易度**: hard（★★★）
> **要求スキル**: ☆1-5 統合——全スキルを使い切る集大成ケース
> **最大ターン**: 15
> **CompanionAI**: トイマル（「……でも、聞いちゃいけないのだ。マコトの誓いが——」。問うことへの内なる葛藤が表面化する段階）

---

## 1. 前提

- `docs/case_dev_guide.md` を必ず先に読むこと
- case_007 は **7番目のケース**。☆1〜☆5全スキルの統合。ここまでで培った全ての「問う力」を駆使する
- 犯人タイプ：**善人型**（温かい優しさそのものが防壁。追い詰めるプレイヤー側が「自分は悪いことをしているのでは？」と感じてしまう）。直前のcase_006は論理型だったので対照的
- 縦軸第3層：**「問わない社会」の受益者が露呈する**。犯人が法的根拠を持って「お前がしていることは犯罪だ」と宣言する衝撃

## 2. 事件の真実

| 項目 | 内容 |
|------|------|
| 舞台 | コトハの街「マコト堂」——街の中心にある教会風の建物。教育機関であり、裁判機関であり、「マコトの誓い」の守護組織。複数の教室、大裁定官の執務室、研究室、古文書庫がある |
| 被害者 | **トイーダ**（男・35歳）——マコト堂の若い教師。ボサボサの黒髪、インク染みの袖口。穏やかだが目の奥に静かな炎。マコト堂の中で唯一「問いの歴史」を密かに研究し、「マコトの誓い」の公式歴史に疑問を持ち始めていた。なのに協力的で、事件の手がかりを何度も提供してくれた存在。名前の由来：「問いだ（問いを発する者）」 |
| 犯人 | **ジオーネ**（男・52歳）——マコト堂の主席教師。「マコト堂のお父さん」と呼ばれる温かく信頼される先生。大柄だが威圧感はなく、安心感を与える。名前の由来：「慈恩」のイタリア風 |
| 動機 | トイーダが「問いの書」の完全な原本を古文書庫の隠し部屋で発見した。この原本は「マコトの誓い」制定以前の聴き役族の技法書であり、「問いの聴き手」としての本来の役割が記されていた。トイーダはこれを公表し、マコトの誓いの真実を明かそうとした。ジオーネはマコト堂の秩序——そして自分自身が依存する「問わない社会」の安定——を守るために、トイーダの口を封じた |
| ジオーネの真実 | ジオーネ自身も若い頃に「なんで？」と問いかけた経験がある。その結果、当時の上層部に呼び出され、右手の薬指を切られた（「問う指を折る」という旧い罰則）。以来、「問わないことが生きること」と学び、優しさで問いを封じる側に回った。**犯人自身も被害者**という構造 |
| 手段 | **窒息（枕による）**。トイーダの研究室を夜に訪ね、「原本について話し合いたい」と持ちかけ、トイーダが机に突っ伏して仮眠していた隙にクッションを顔に押しつけた。「研究で無理をして、机に突っ伏したまま心臓が止まった」——過労死に偽装 |
| 偽装 | トイーダは連日の徹夜研究が周囲に知られていた。目の下のクマ、インク染みの服。「研究に没頭しすぎた」という過労死の説明は自然に受け入れられた |

## 9. 感情的アンカー

トイーダのノートの最後の一行。「問いは罪ではない。問いは——」。途切れた言葉。この続きは永遠にわからない。

introで登場（途切れた一行）→ victoryで回収（「その問いは、ここにいる全員に引き継がれた」）。

---

## 10. トイマルの特別な体験（最重要）

このケースはトイマルの内なる葛藤がピークに達する。

「問いの書」の原本が発見されたとき、トイマルの種族の真実——「聴き役」族はかつて「問いの聴き手」だった——が判明する。トイマルはこれに激しく動揺する。

> **トイマル**「……ボクの種族は、昔は『問う者の傍で記録する者』だったのだ？ 問いを……助ける側だったのだ？」
> **なの**「トイマル……」
> **トイマル**「……でも、聞いちゃいけないのだ。マコトの誓いが——。ボクは、問っちゃいけないのだ。ボクは——」

**これはcase_008のトイマル覚醒への直接の布石。** ここではまだ覚醒しない。「問いたいけど、問えない」という引き裂かれた状態で終わる。

---

## 11. 縦軸ストーリー要素（第3層）

- マコト堂の内部で事件が起きる→制度の中枢で不正が行われている
- ジオーネの崩壊時台詞「お前がしていることは、マコトの誓いに反する犯罪だ」——これは強がりではなく**法的に正しい発言**。なのの「問う」行為はこの世界では実際に犯罪
- なのが気づく：「この人、嘘がうまいんじゃない。周りが誰も聞かないだけだ。……いや、それだけじゃない。問うことが違法なら、嘘は制度に守られてるんだ」
- 大裁定官の側近が圧力をかけていた事実→権力が「問わない社会」を積極的に維持している
- 「問いの書」の原本→聴き役族の真実→トイマルのアイデンティティ危機
