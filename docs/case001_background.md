## 1. 舞台：ひだまり動物病院

### 外観

住宅街の角地に建つ木造モルタル2階建て。築30年以上だが手入れが行き届いている。

- **壁**：白いモルタル。ところどころ蔦（つた）が這っている
- **看板**：オレンジ色の温かみのあるプレート型。「ひだまり動物病院」の文字。犬と猫のシルエットのロゴ入り。夜は看板灯が点灯し、暖色の光で文字が浮かぶ
- **正面入口**：ガラスの自動ドア。右横に暗証番号式のテンキーロック。ドアの上部に防犯カメラ1台（入口正面を撮影）
- **外壁のポスター**：「日曜日は動物ふれあい教室！」の手書きポスター。子どもが描いた犬猫の絵付き
- **周囲**：住宅街の静かな通り。隣家（田中ハルさん宅）とは8メートルほどの間隔。病院前に自転車置き場。最寄りのコインパーキングは徒歩2分
- **夜の外観**：看板灯のオレンジ、2階窓に差し込む街灯の光。住宅街の静寂。虫の音

### 1階フロア

#### 待合室
- 入口を入ってすぐの空間。白壁にクリーム色のタイル床
- **壁面**：子どもたちが描いた動物の絵が画鋲で留めてある（犬、猫、うさぎ、ハムスターなど。クレヨン画）
- **窓際**：陽当たりの良い出窓。クッション（深緑のコーデュロイ素材、へたっている）＝猫のハチの定位置
- **椅子**：木製のベンチ3脚。座面に薄いクッション
- **受付カウンター**：低めの木製カウンター。上にベル、ペン立て、パンフレット類。カウンター奥に「本日の担当」ホワイトボード
- **天井**：蛍光灯。夜間は消灯（入院室の常夜灯のみ）
- **重要構造**：正面入口から入ると必ず待合室を通る。待合室の奥に1階の各部屋と2階への階段がある。他に入口はない

#### 入院室
- 待合室の奥、右手のドア。動物の鳴き声が漏れないよう防音ドア
- **ケージ**：ステンレス製の動物用ケージ10台（壁面に2段×5列）
- **照明**：夜間は青白い常夜灯のみ点灯
- **床**：防水のリノリウム。排水口あり
- **設備**：点滴スタンド、体温計、投薬用シリンジ、記録ボード
- **汚れ物入れ**：大型のランドリーバッグ。使用済みタオル、汚れた白衣を入れる。犯人はここに血のついた白衣を紛れ込ませた
- **事件の夜の入院動物**：犬2頭（ゴールデンレトリバーのクッキー、柴犬のコタロウ）、猫1匹（老猫のタマ・三毛猫・おとなしい性格）

#### 診察室（2室）
- 第1診察室：犬飼先生の主な診察室。ステンレスの診察台、棚に医療器具
- 第2診察室：主に遊上が補助に入る部屋

#### 手術室
- 1階奥。ステンレス手術台、無影灯、麻酔器

#### 薬品庫
- 施錠式。鍵は犬飼先生と遊上の2人だけが所持

### 2階フロア

#### 階段
- 待合室の奥から上がる直線階段。木製。手すり付き。12段
- **踊り場**：階段を上がりきった先の踊り場。ここに犬飼先生の遺体が偽装配置された
- 踊り場から左手に院長室のドア、右手に倉庫のドア

#### 院長室（事件現場）
- **ドア**：木製の引き戸。鍵なし
- **入口から見て正面**：窓。住宅街に面する。カーテンは薄手のレース。夜は街灯の光がオレンジ色に差し込む
- **入口から見て左手**：獣医学書の本棚。天井近くまでの大型書棚。分厚い専門書が並ぶ。最上段に犬飼先生の卒業アルバム。中段に凶器となった「獣医学大辞典」（厚さ12cm、重さ約4kg、紺色の背表紙に金文字）
- **入口から見て右手**：犬飼先生のデスク。木製。年季が入っている。デスクの上に家族写真（犬飼先生と妻。笑顔）と「動物ふれあい教室」の集合写真（先生の膝の上にハチ、周りに子どもたち8人ほど）。デスクランプ（緑色のバンカーズランプ）。文具入れ、日めくりカレンダー
- **壁**：動物学会の修了証、感謝状。子どもからの手紙（「犬飼先生ありがとう」のクレヨン画つき）
- **床**：フローリング。古くてところどころ軋む
- **夜の院長室の光景（重要）**：照明を消すと、窓から街灯のオレンジ色の光が差し込み、本棚と犬飼先生のデスクがシルエットで浮かぶ。昼間とはまったく違う、静かで親密な雰囲気。この光景を知っていることが弱点1の核心

#### 倉庫兼会議室
- 段ボール、季節の装飾品（クリスマスの飾り等）、古い医療機器

### 環境音

- **昼**：犬の鳴き声、受付のベル、子どもたちの笑い声、犬飼先生の穏やかな声
- **夜**：入院動物の寝息、ポンプ式給水器の音、遠くの車の音、虫の声
- **事件の夜**：22:28に遊上の叫び声「なんで…先生…なんで逃げるんだよ…」

## 2. キャラクター

### 犬飼 善（いぬかい ぜん）── 被害者

| 項目 | 内容 |
|------|------|
| 年齢 | 62歳 |
| 職業 | ひだまり動物病院 院長（獣医師） |
| 家族 | 妻あり。子どもなし |
| 名前の由来 | 「犬を飼う善い人」── 人柄をそのまま名前にしたダジャレ |

#### 外見（画像生成用詳細）
- **体格**：中肉中背。やや猫背。歳の割に手先がしっかりしている
- **顔立ち**：丸顔。目尻に深い笑いジワ。穏やかな垂れ目。額が広い
- **髪**：白髪混じりのグレー。短めの七三分け。つむじが目立つ
- **眼鏡**：丸い銀縁のボストン型メガネ。度が強く、目が少し小さく見える。事件後、階段の踊り場で割れた状態で発見
- **服装（白衣姿）**：清潔だがやや着古した白衣。胸ポケットにボールペン2本と聴診器。**ポケットには常に動物用ビスケット**が2〜3個入っている
- **服装（私服）**：チェック柄のネルシャツ、コーデュロイのパンツ、ニューバランスのスニーカー。カーディガン
- **手**：大きく、温かそう。指先に細かい傷（動物の爪跡や噛み跡の古傷）
- **持ち物**：古い革のカバン。中に日記帳

#### 性格・人柄
- 口癖は**「動物を助けるのに理由はいらんよ」**
- 子どもに話すときは必ずしゃがんで目線を合わせる
- 毎晩帰る前にハチのケージの前で**「ハチ、おやすみ」**と声をかけるのが日課
- 毎週日曜日に「動物ふれあい教室」を開催。地域の子どもたちが参加
- 人を信じる力が強い。遊上を前職のトラブル込みで採用した。「この子は大丈夫だ」と信じた
- 最近、遊上の様子がおかしいことに気づいていた。日記に不安を記していた
- 最後の言葉：**「蓮くん、それは間違っている」**

#### 感情的意味
- 犬飼先生は「純粋な愛」の象徴。動物にも人にも無条件の愛を注ぐ人
- その善良さゆえに、遊上の歪みを「話せばわかる」と信じて夜の病院に来た
- 犬飼先生の不在が事件後の病院に「空洞」を作る。その空洞をプレイヤーが感じること＝事件を解決する動機

### 遊上 蓮（ゆがみ れん）── 犯人

| 項目 | 内容 |
|------|------|
| 年齢 | 27歳 |
| 職業 | ひだまり動物病院 獣医助手（夜勤担当） |
| 出身 | 地方出身。東京の動物専門学校卒（獣医師免許は未取得） |
| 名前の由来 | 「歪み」＝歪んだ人格。「蓮」＝蓮の花（泥の中から美しく咲くが根は汚泥に絡む） |

#### 外見（画像生成用詳細）
- **体格**：175cm前後。細身だが肩幅はしっかり。白衣が映えるスタイル
- **顔立ち**：整った顔立ち。切れ長の目。鼻筋が通っている。唇が薄い。笑うと口角が上がり、非常に好印象。女性にモテる顔
- **髪**：黒髪の短髪。さわやかなマッシュショート。前髪は眉にかかるくらい。清潔感がある
- **目**：通常は明るく澄んだ目。だが犬飼先生の話題になると一瞬だけ奥の光が変わる──熱を帯びた、独占欲の色
- **服装（白衣姿・通常）**：真っ白でシワのない白衣。犬飼先生の着古した白衣との対比。胸ポケットにペンライトとメモ帳。白衣の下は無地のTシャツとスラックス
- **服装（崩壊時）**：白衣の前ボタンが外れている。髪が乱れている。左腕に赤いひっかき傷（犬飼先生の爪跡）
- **手**：きれいな手。長い指。動物を優しく扱う手つき
- **表情のバリエーション（重要）**：
  - **平常時**：柔らかい笑顔。人懐っこい目。信頼を誘う表情
  - **犬飼先生の話題**：一瞬だけ目の奥に炎が灯る。本人は気づいていない
  - **動揺時**：笑顔が引きつる。目が泳ぐ。額に汗
  - **崩壊時**：目が見開かれる。口元が歪む。涙。「僕」が「俺」に変わる瞬間の表情

#### 経歴・バックストーリー
- 幼少期に両親が離婚。祖母に育てられたが中学時代に祖母が他界
- 「自分を無条件に受け入れてくれる人」への渇望を抱えて育った
- 容姿端麗で女性にはよくモテるが、本人は誰にも本気で心を開けない
- 前職のペットショップで客とトラブルを起こし退職。行き場をなくしていた
- 犬飼先生に「うちで働いてみるか」と拾われる。3年前
- 犬飼先生だけが「蓮くん」と名前で呼び、叱りもするが見捨てなかった
- 遊上の中で犬飼先生は「世界で唯一、自分を受け入れてくれた人」になった
- その感情はやがて歪み、「先生を自分だけのものにしたい」独占欲に変わった
- 匿名の手紙を送り、先生の行動を把握し、二人きりになれる夜勤を志願した
- **遊上はこの感情が「間違っている」と気づいていない。「深い恩義と感謝」だと信じている**

#### 性格の3段階（ゲーム内の口調変化）

**① 平常時（コヒーレンス 80〜100）**
- 一人称「僕」。柔らかい敬語。笑顔が自然
- 口癖：「あ、はい、もちろんです」「犬飼先生にはお世話になっていて」
- 聞かれていないことまで「ちなみに〜」と補足する
- 犬飼先生の話だけ、無意識に熱がこもる

**② 動揺時（コヒーレンス 40〜79）**
- 「えっと…」「いや、それは…ちょっと」が増える
- 犬飼先生の話で話を逸らそうとする
- 笑いで誤魔化す。語尾が断定を避け始める
- 犬飼先生を他者が親しげに呼ぶと表情が硬くなる（嫉妬の片鱗）

**③ 崩壊時（コヒーレンス 0〜39）**
- 「僕」が「俺」に変わる
- 「…うるさいな」「だから何だよ」と攻撃的な短文
- 直後に「…すみません」と取り繕う
- 犬飼先生の話で声が震える
- **最後の崩壊：「先生は僕だけのものだったのに」── 自分の言葉に自分で驚く**

#### キャラクターのテーマ
- 遊上は「完全な悪人」ではない。動物への愛情は本物。犬飼先生への感謝も本物
- しかし愛情の形が致命的に歪んでいる。「守りたい」が「所有したい」にすり替わっている
- 本人がそのことに気づけないまま犯行に至り、崩壊の瞬間に初めて自分の歪みに直面する
- **「歪んだ愛」が最愛の人を破壊する悲劇**

### ハチ ── 感情的アンカー（猫）

| 項目 | 内容 |
|------|------|
| 種類 | 三毛猫（メス） |
| 年齢 | 推定4歳 |
| 特徴 | 片耳がない（右耳が根元から欠損。台風の夜に負傷した状態で保護された） |
| 来歴 | 3年前の台風の夜、犬飼先生が保護。「ハチ」と命名 |

#### 外見（画像生成用詳細）
- **毛色**：白をベースに、茶色と黒の三毛柄。胸元は白。背中に大きな茶色の斑
- **体格**：標準的なメス猫。やや小柄
- **右耳**：根元から欠損。傷跡はきれいに塞がっている。片耳がないことでシルエットが特徴的
- **目**：大きな黄緑色の目。穏やかだがどこか物悲しい
- **定位置**：待合室の窓際の深緑のコーデュロイクッションの上。日向で丸くなっている姿
- **仕草**：来院する子どもに撫でられるのが好き。犬飼先生が帰るとき、入口まで見送りに来る

#### 物語上の役割
- **ハチ＝犬飼先生に「純粋に」愛された命**
- 遊上も同じく犬飼先生に拾われた存在。しかし遊上の愛は歪み、ハチの信頼は純粋
- 同じ「先生に救われた命」でありながら、片方は純粋な信頼、片方は破壊──この対比がケースの感情的背骨
- **intro**：ハチが院長室のドアの前で待っている。先生の「おはよう」が来ない
- **victory**：割れた眼鏡のそばのビスケットを、ハチがひとつだけくわえて窓際のクッションに戻る

## 4. カラーパレット・ビジュアルトーン

### 病院（昼）
- 温かい暖色系。白壁にオレンジの看板。陽だまりの光
- 子どもの絵、木の家具、クッションの深緑。生活感と安心感
- キーカラー：**オレンジ、クリーム、深緑**

### 病院（夜・事件）
- 暗く、静か。入院室の青白い常夜灯。院長室に差し込む街灯のオレンジ
- 日常の温かさが消え、影が深くなる。同じ場所なのに別の顔
- キーカラー：**暗いオレンジ、影のブルーグレー、黒**

### 遊上蓮
- 平常時：清潔な白（白衣）、黒（髪）。明るくさわやか
- 崩壊時：白衣が乱れる。影が顔に落ちる。目の奥に暗い炎
- キーカラー：**白→暗転**

### 犬飼先生
- 穏やかな暖色。着古した白衣、チェックのシャツ、丸い眼鏡
- 温かさと安心感。不在になったとき、その「色」が消える
- キーカラー：**オフホワイト、ブラウン、オレンジ**

### ハチ
- 白・茶・黒の三毛。片耳のない独特のシルエット
- 犬飼先生の「温かさ」を体現する存在。先生がいなくなった病院に残る最後の温度
- キーカラー：**白、アンバー、墨**

## 5. キーヴィジュアル案

### メインビジュアル
「夜の動物病院の正面。看板灯のオレンジの光。2階の窓からうっすら街灯のオレンジが見える。1階の窓際に、片耳のない三毛猫のシルエット。手前に遊上蓮の横顔（白衣、整った顔、目だけが暗い）。奥にぼやけた犬飼先生の笑顔の残像」

### 事件発見シーン
「朝。階段の踊り場に倒れた犬飼先生。割れた丸い眼鏡。散らばった動物用ビスケット。2階の院長室のドアの前に座る片耳の三毛猫。朝日が窓から差し込んでいるが、院長室だけ暗い」

### 尋問シーン
「遊上蓮の正面アップ。柔らかい笑顔。白衣。背景は病院の待合室。ただし目の奥だけが暗く、不穏」

### 崩壊シーン
「遊上蓮が膝をつき、両手で顔を覆っている。白衣が乱れ、左腕のひっかき傷が見えている。背景は暗転した院長室。窓からのオレンジの街灯だけが光源」

### エンディング
「待合室の窓際。夕暮れの光。深緑のクッションの上で丸くなるハチ。そのすぐ横に、犬飼先生の動物用ビスケットが1つだけ置かれている。誰もいない待合室」

## 6. 物語のテーマ構造

```
犬飼先生（純粋な愛）────── 対比 ──────遊上蓮（歪んだ愛）
  ↓                                       ↓
動物を無条件に助ける              先生を無条件に所有したい
遊上を拾って育てた                先生に拾われて依存した
「それは間違っている」             「なんで逃げるんだよ」
  ↓                                       ↓
殺される                          殺す
  ↓                                       ↓
ハチだけが残る                     自分の歪みに気づく
（純粋な愛の残響）                 （崩壊の瞬間）
```

**名前の対比：**
- **犬飼 善**（いぬかい ぜん）＝犬を飼う善い人。名前が人格そのもの
- **遊上 蓮**（ゆがみ れん）＝歪んだ人格。蓮は泥の中から咲く美しい花だが、根は泥に絡む

クリア後に名前のダジャレに気づいたとき、もう一度物語の意味が変わる。
